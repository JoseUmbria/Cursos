Resumen curso de Git / GitHub

Git es un software de control de versiones de un proyecto que facilita su desarrollo y mantenimiento de manera individual o como trabajo en equipo. Guarda un registro de todos los cambios y se puede recuperar información anterior si es necesario.

Es posible trabajar con múltiples archivos, además de permitir el trabajo de varias personas al mismo tiempo. Para ello se usan las ramas, que posteriormente se fusionan con merge.

El programa GIT se debe instalar y posteriormente se trabajará con GIT BASH que es la consola donde se desarrollarán todas las funciones.

En GIT hay dos áreas que son el área de ensayo y el repositorio local. La primera es temporal y se puede ver si el fichero tiene seguimiento de cambios por GIT, la segunda es la copia local del archivo.

Funciones importantes de GIT

git init -> Se ejecuta sólo una vez al empezar el proyecto o empezar el seguimiento del proyecto.
